# Description
Convert3d is a online 3d model converter for faster, easier file conversions. This site is meant to be functional for real life purposes if someone wants to convert a file for 3d printing. It is an open source site anyone can use with no download or anything! I will be adding more file type supports in the future.

# Code
Some of the hard code was made by copilot but I had to tweak a bit and give some structure. It is made of html for structuring, css for the cool visual enhancements, javascript so the site can communicate and be interactive, python for the upload backend server, and it is all connected to a nest server to run 24/7.
